# ImanTap Bot — Описание проекта

## Что это такое

**ImanTap** — Telegram-бот для мусульман, помогающий отслеживать духовные практики в Рамадан и Шавваль. Бот уведомляет о времени намаза, помогает отслеживать ибадат (намаз, пост, Коран, зикр), начисляет очки опыта (XP) за активность и объединяет пользователей через реферальную систему.

Вместе с ботом работает **Mini App** (веб-фронтенд внутри Telegram) — через него пользователи логируют свой прогресс, бот же занимается уведомлениями, платежами и настройками.

---

## Стек

- **Runtime:** Node.js 20.x, ES modules (`import`/`export`)
- **Telegram:** `node-telegram-bot-api` (polling, не webhook)
- **БД:** MongoDB через официальный драйвер `mongodb`
- **Планировщик:** `node-schedule`
- **Геолокация:** `geo-tz`, `node-geocoder`, Nominatim OSM API
- **Деплой:** Railway.app (`nixpacks.toml`)

## Основные функции

### Онбординг
1. Сбор номера телефона
2. Выбор города / геолокация
3. Ввод промокода (опционально, даёт скидку 500 тг)
4. Выбор оплаты или демо-режима (24 часа)

### Времена намаза
- **Основной источник:** ҚМДБ (api.muftyat.kz) — для Казахстана
- **Запасной:** Aladhan API — для других стран
- Пользователь может переключать источник в настройках
- Ежедневное обновление в 00:30 UTC

### Уведомления
- Напоминание за 15 минут до Фаджр (сухур) и Магриб (ифтар)
- Ежедневное напоминание логировать прогресс (20:00 UTC)
- Уведомления об истечении подписки (за 3 дня, за 1 день, в день)

### Оплата и подписка
- Стоимость: 2 490 тг (с промокодом — 1 990 тг)
- Подписка на 90 дней
- Пользователь отправляет фото чека → администратор одобряет/отклоняет
- При отклонении выдаётся демо-режим (один раз)

### XP и геймификация
- XP за: намаз (50), таравих (100), пост (200), Коран (100), зикр и т.д.
- Мультипликатор стрика: от 1.0x до 3.0x
- Реферальный XP: 100 XP за регистрацию, 400 XP за оплату реферала
- Мультипликатор рефералов по дневному количеству (1.3x / 1.6x / 2.0x)
- Лидерборды: глобальный, среди друзей, по городу/стране

### Кружки (circles)
- Пользователи создают группы до 10 человек
- Вступление по инвайт-коду
- Совместное отслеживание прогресса

## Модель пользователя (MongoDB)

Коллекция `users`. Ключевые поля:

```js
{
  userId,           // Telegram ID (unique index)
  language,         // 'kk' | 'ru'
  location: { city, country, latitude, longitude, timezone },
  prayerTimes: { fajr, sunrise, dhuhr, asr, maghrib, isha, source },
  paymentStatus,    // 'unpaid' | 'paid' | 'demo' | 'subscription_expired'
  subscriptionExpiresAt,
  promoCode,        // Уникальный 6-символьный реферальный код
  referredBy,
  xp, currentStreak, longestStreak,
  progress,         // { day: { taskKey: bool } }
  dailyGoalRecords, // Прогресс по категориям целей (v2)
  shawwalFasts, shawwalDates,
  prayerTimeSource, // 'muftyat' | 'aladhan'
}
```

## Архитектурные особенности

- **Polling** — бот опрашивает Telegram API, не требует публичного домена для webhook
- **In-memory сессии** — сбрасываются при перезапуске (приемлемо для онбординга)
- **Атомарность XP** — защита от двойного начисления через массивы `earnedTasks` и флаги `earnedGoalXp`
- **Таймзоны** — у каждого пользователя хранится свой timezone, все уведомления вычисляются в его локальном времени
- **Языки** — бот полностью двуязычный (казахский / русский), все строки выбираются по полю `user.language`
